Java1.7-1.8-新特性
Created 2017-07-28| Updated 2019-12-26|Java
Post View:

JDK 1.7

  • 所有整数可用二进制表示，二进制前加0b

  • switch支持String

  • try-with-resourse语句:实现了AutoCloseable接口的资源就能使用该结构，try中语句结束之后会自动关闭释放资源。

  • Exception并列:不是包含关系的异常可并列捕获

  • 泛型可通过类型推导简化，省略<>后面的部分。
